# Contexto general de proyectos

Este directorio contiene los repositorios activos de Ibuprofeno Fernández, organizados por dominio.

## Repositorios

| Directorio | Dominio |
|---|---|
| `sistemas/` | Diseño e ingeniería de sistemas |
| `musica/` | Composición y producción musical |
| `vinilos/` | Escritura y crítica discográfica |
| `charlas/` | Preparación de charlas y presentaciones |

Cada repositorio tiene su propio `CLAUDE.md` en `.claude/` con contexto específico y estado de sesión.

---

## Convenciones comunes

- Markdown como formato base para documentación
- Diagramas en PlantUML cuando aplica
- Código en el lenguaje apropiado para cada dominio (ver `CLAUDE.md` de cada repo)
- Documentación siempre en el idioma del proyecto

## Instrucciones operativas

- Verificar el estado real de cada fichero antes de asumir que está completo.
- Al completar una sesión, actualizar la sección **Estado de sesión** del `CLAUDE.md` del repo correspondiente.
