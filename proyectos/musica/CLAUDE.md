# musica — Composición y producción

## Contexto del proyecto

Repositorio de composiciones, experimentos sonoros y proyectos de producción musical de Ibuprofeno Fernández. Incluye partituras, patches de síntesis y notas de producción.

## Estructura

```
musica/
├── composiciones/      ← partituras y MIDI
├── produccion/         ← notas de sesión, cadena de señal
├── experimentos/       ← prototipos sonoros descartables
└── referencias/        ← análisis de obras de referencia
```

## Convenciones

- Las notas de producción documentan la cadena de señal completa: instrumento → procesado → mezcla
- Los experimentos no se borran: se archivan en `experimentos/descartados/` con una nota de por qué no funcionaron
- Las referencias incluyen siempre el pressing concreto analizado (sello, año, país, matriz)
